\chapter{结论与展望}

\section{研究结论}

本文对基于充气结构的车载多形态飞行机器人系统（飞行雨伞）进行了从概念设计、理论计算到样机制作和实验验证的全部工程过程。主要结论如下：

\textbf{（1）充气结构可以实现飞行器形态的主动重构，工程上可行。}
用快递充气气柱做骨架材料，配合 PLA 3D 打印节点和 TPU 薄膜伞面，系统完成了收纳态到飞行遮雨态的主动切换。伞面正投影遮蔽面积为 $0.608\,\text{m}^2$，整骨架充气展开时间为 43\,s，气密性保持时间大于 30\,min，排气收纳时间为 48\,s，整个工作闭环经过实验端到端的测试。

\textbf{（2）充气伞形多旋翼系统可以保持稳定的悬停飞行。}
根据对角旋翼轴距 1230\,mm 的正方形 X 型四旋翼布局，使用 SIRIUS 2306.5-1850KV 电机与 51366 MCK v3 桨叶，整机实测起飞质量为 1683\,g，实际悬停油门约为 43\%，推重比为 1.68，系留模式实测续航为 18.2\,min，各动力指标满足设计需求。针对系统转动惯量约为同质量标准四旋翼 17 倍的特性，本文采用将比例增益降至 PX4 默认值 $1/3$ 的 PID 初始整定策略，实飞验证中取得了 Roll 轴 $\pm2.1^\circ$、Pitch 轴 $\pm1.8^\circ$ 的悬停精度。

\textbf{（3）车载快速部署方案在工程上可行。}
系统以车载微型充气泵充气，零改装集成于汽车后备箱或座椅背后，
从取出到完成起飞的准备时间 85\,s，
完整工作流程（收纳→展开→飞行→收纳）得到端到端验证。

\textbf{（4）两种供电形态覆盖了不同使用需求。}
自载电池模式（6S 1500\,mAh）用于应急备用与短时功能验证；
系留供电模式（6S 10000\,mAh）通过地面有线供电实现长时悬停，
不受机载电池容量限制。两种模式插拔切换，互为补充。


\section{创新点总结}

\begin{enumerate}
    \item \textbf{充气结构作为飞行器形态主动重构的核心手段}：
          将薄壁充气气柱引入多旋翼飞行器的形态重构设计，
          以充气/排气驱动系统在紧凑收纳态与大面积飞行遮蔽态之间主动切换。
          相比刚性折叠方案，充气方案可展开的功能结构面积量级更大，
          且收纳后体积接近零，较好地化解了大面积遮蔽与紧凑车载收纳之间的矛盾。

    \item \textbf{面向充气伞形载体的多旋翼布局专项设计}：
          针对方形充气伞体面积大、气动阻力显著、转动惯量远超常规四旋翼的特性，
          开展了旋翼布局、推重比设计与 PID 整定的专项研究，
          验证了超大转动惯量场景下稳定悬停飞行的可行性。

    \item \textbf{车载部署与个人出行服务的系统化方案}：
          将飞行机器人定位为"车载服务系统的延伸执行器"，
          提出涵盖收纳固定、快速充气展开、离车悬停服务与归位回收的完整车载部署方案。
          这一应用定位在现有车载无人机研究中属于较新的方向。
\end{enumerate}


\section{不足与局限}

受样机阶段工程条件与时间限制，系统在以下方面存在不足：

\begin{itemize}
    \item \textbf{自主跟随能力缺失}：
          当前系统依赖操作者手动遥控，不具备对用户的自主跟随能力。
          视觉目标检测与跟随控制是这一功能的前提，
          也是从样机走向可用产品需要突破的核心问题。

    \item \textbf{悬停精度与抗风能力有限}：
          无 GPS 辅助条件下，位置保持依赖操作者手动修正。
          充气伞面受风面积大，较强侧风下悬停精度难以保证，
          对准用户头顶的难度随风速增大而上升。

    \item \textbf{遮雨功能未进行实测验证}：
          受实验条件限制，本文未在实际降雨环境下开展飞行状态的遮雨测试。
          伞面防水性能目前仅基于 TPU 材料特性与遮蔽面积的理论推断，
          实际降雨场景下的遮护效果有待验证。

    \item \textbf{充气骨架节点刚度有限}：
          当前骨架使用商业快递气柱，节点连接刚度偏低，
          受力或侧风下存在明显形变。
          定制截面气柱或与刚性件的混合结构或可改善这一问题。

    \item \textbf{动力效率低于预期}：
          实测整机质量（1683\,g）超出预算约 14.6\%，
          加之 51366 桨叶在低转速下的气动效率低于理论估计，
          实际悬停油门（43\%）高于设计目标（38\%）。

    \item \textbf{安全性设计尚不完善}：
          飞行器悬停于用户头顶，对失效模式的处理要求较高。
          当前样机的失控保护与机械安全结构仅完成初步设计，
          距产品级安全标准仍有差距，后续需开展 FMEA 分析与冗余设计。
\end{itemize}


\section{未来工作展望}

后续研究可从以下四个方向推进：

\textbf{方向一：自主视觉跟随}

在飞行雨伞平台上集成轻量化计算模块（NVIDIA Jetson Nano），
结合单目或深度摄像头和目标检测算法（YOLOv8-Nano），
实现对用户的实时识别、深度估计和跟踪控制，
可以使系统能够在无人工干预情况下跟随用户步行移动。

\textbf{方向二：结构刚度与轻量化的协同优化}

定制截面高压气柱与碳纤维骨架的混合结构方案，
在保持折叠收纳特性的同时提升骨架刚度和抗侧风能力；
用通过拓扑优化减小 PLA 节点质量，降低整机起飞质量。

\textbf{方向三：低噪声旋翼系统优化}

研究低桨盘载荷旋翼方案（更大直径、更低转速），
结合旋翼声学优化设计，将近人飞行的噪声控制在可接受范围内
（目标 $\leq 65$\,dB @ 1\,m）。

\textbf{方向四：车载一体化平台产品化}
设计专用车载底座，集成自动充气泵、快速展开机构与无线充电功能，
探索飞行雨伞作为新能源汽车生态配件的产品化路径。

\vspace{1em}
回到本文题目中的"车载 Agent"：飞行雨伞在本文中实现的是
物理执行层面的自主性——响应指令、自主起降、完成遮雨任务；
感知层（视觉跟随）与认知层（意图理解）尚未涉及。
本文的工作止步于执行层的工程验证，
上述四个方向的推进将逐步补全感知与决策两个层次的能力，
使系统向更完整的 Agent 架构演进。
